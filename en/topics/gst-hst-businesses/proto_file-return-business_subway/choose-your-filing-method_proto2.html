Mandatory electronic filing

If you are a GST/HST registrant with a reporting period that begins in 2024 or later, you must file your returns electronically (except for charities and selected listed financial institutions). If you don't, we may charge you a penalty.

Read more about mandatory electronic filing and exceptions

Filing methods: out of

  • My Business Account (using NETFILE)

    File your GST/HST return from within your CRA My Business Account.

    Benefits
    Limitations

    Learn how to file: Using My Business Account

  • Represent a Client (using NETFILE)

    Registered representatives authorized with access level 2 and legal representatives can file GST/HST returns for their clients in Represent a Client.

    Benefits
    Limitations

    Learn how to file: Using Represent a Client

  • [NETFILE form]

    If you do not have access to My Business Account or Represent a Client you can file your GST/HST return directly with the CRA using the NETFILE form online.

    This filing method requires an access code.

    Benefits
    Limitations

    Learn how to file: Using the NETFILE form with an access code

  • Third-party accounting software

    You can file your GST/HST return directly to the CRA using third-party accounting software through the internet file transfer method.

    Benefits
    Limitations

    Learn how to file: Using third-party accounting software

  • Through a financial institution

    You can file your GST/HST return online or in person at a participating Canadian financial institution via electronic data interchange (EDI).

    Benefits
    Limitations

    Learn how to file: Through a financial institution

  • By phone

    You can file your GST/HST return by phone using CRA's TELEFILE service.

    Benefits
    Limitations

    Learn how to file: using TELEFILE

  • By mail

    You can file by mail by sending your paper GST/HST return (GST34-2 or GST62 or RC7200?) to the address on the return

    You may be charged a penalty if you file by mail, unless you are a charity, a selected listed financial institution, or meet another exception.

    Read more about mandatory electronic filing and exceptions

    Benefits
    Limitations

    Learn how to file: By mail
