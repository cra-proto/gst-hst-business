The GST/HST access code is a four-digit access code is required for some of GST/HST filing methods. It acts as a personal security key that authenticates you when you are not logged into a CRA account, and should not be shared with unauthorized individuals.

A representative that is legally authorized to sign the paper copy of your GST/HST return, may also use your access code to file on your behalf using the methods that require an access code.

Your GST/HST access code will remain the same for all filing periods unless you change it.

When an access codes is required

CRA's sign-in services such as My Business Account and Represent a client do not require the use of a GST/HST access code, they are more secure and provide additional services.

Is the GST/HST access code a different code than the access code used for individual filing? If not, something should be explained about it here.

Methods that do not require an access code:

  • My Business Account / Rrepresent a client
  • GST/HST Electronic Data Interchange (EDI)
  • Paper filing

Methods that require an access code:

  • GST/HST Internet File Transfer
  • GST/HST TELEFILE
  • GST/HST NETFILE (only when accessed though the web form/application)

Obtaining an access code

Explain how an access code is sent to registrants when they register?

  • If you are a new user and have never received an access code, call 1-800-959-5525.
  • If calling from outside Canada and the United States, call collect at 613-940-8497.

Changing an access code

GST/HST Access Code Online service allows business owners to change their access code to a number of their choice. The new access code can be used to file all future returns.

  • To change your GST/HST access code, you will need to enter your business number and your choice of the following information from a previously filed return:
    • the reporting period and line 109 (net tax) where the amount on line 109 is not $0.00
    • the reporting period and confirmation number from a return filed using GST/HST NETFILE, TELEFILE, or Internet File Transfer
  • The access code can also be created or changed over the phone by the business owner or an authorized representative with level 2 authorization or higher.
