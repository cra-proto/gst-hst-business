Mandatory electronic filing

If you are a GST/HST registrant with a reporting period that begins in 2024 or later, you must file your returns electronically (except for charities and selected listed financial institutions). If you don't, we may charge you a penalty fee.

Learn more: Mandatory electronic filing requirements and exceptions

CRA account

This filing method does not require a GST/HST access code.

If you already know how to file, you can start filing by signing in to your CRA account.

NETFILE form

This filing method requires a GST/HST access code.

If you already know how to file, you can start filing your return with NETFILE.

Financial Institution

This filing method does not require a GST/HST access code.

Third-party software

This filing method requires a GST/HST access code.

If you have already completed your return, you can upload your return through the GST/HST Internet File Transfer.

TELEFILE

This filing method requires a GST/HST access code.

Mail

This filing method does not require a GST/HST access code.

Possible penalty if you file by mail

You may be charged a penalty fee if you file by mail, unless you are a charity, a selected listed financial institution, or meet another exception.

You may need an access code before you file

If you plan to file by NETFILE, third party software, or TELEFILE, you will require a 4-digit access code.

Learn when you may need and how to get a GST/HST access code.

How to file

When you have chosen your method, learn how to file.

In your online CRA account

File online in your CRA account (My Business Account or Represent a Client)

Using the online form with an access code

File online using the NETFILE form with an access code

Through a financial institution

File through a Canadian financial institution through Electronic Data Interchange

Using third-party software

File with your approved accounting software using Internet File Transfer

By phone

File electronically by phone using TELEFILE

By mail

File a paper return if you are unable to file electronically
